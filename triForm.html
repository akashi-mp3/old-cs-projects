Works in conjunction with triForm.py, which must reside in same directory as this file (triForm.html) on a web server.

Usage:

Enter the numerical value of a side of the the triangle you want to find the area of in the text field, click submit.
Your browser will use this data to form a new request to the server, triggering triForm.py, which will respond with a new page with the perimeter and area of the given triangle.
