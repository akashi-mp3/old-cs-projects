Works in conjunction with textForm.py, which must reside in same directory as this file (triForm.html) on a web server.

Usage:

Enter some text, select what you want to do to it (ROT13 or Reverse), check out the checkboxes (bold, underline, italic) you want, then click submit.
Your browser will use this data to form a new request to the server, triggering textForm.py, which will respond with a new page with text.
